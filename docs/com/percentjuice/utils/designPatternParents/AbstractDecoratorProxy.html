Classpublic class AbstractDecoratorProxy
InheritanceAbstractDecoratorProxy flash.utils.Proxy

extend this class to facilate DecoratorProxy use. use DecoratorProxy when child props are either undetermined or overly numerous (as when decorating a Sprite). ex: if this has public var name, calling this.name returns this.name whereas calling this.blah returns child.blah. you'll want to assure that child contains prop blah. your IDE may throw Errors unless using dynamic syntax such as: trace(proxyInstance["thePropery"]); trace((proxyInstance as Object).theProperty);
