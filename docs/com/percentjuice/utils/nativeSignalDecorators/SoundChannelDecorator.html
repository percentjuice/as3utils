Packagecom.percentjuice.utils.nativeSignalDecorators
Classpublic class SoundChannelDecorator
InheritanceSoundChannelDecorator Object

Utility for common SoundChannel methods though no support for queuing. handles sound firing in rapid, overlapping succession. allows for organized listening to any consecutive SoundComplete Signal. allows for multiple overlapping loops. all loops are tied to one kill switch.
